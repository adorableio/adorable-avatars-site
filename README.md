# adorable-avatars-site

### Includes
* Node
* Express
* Bower
  * jQuery
  * Ember
  * Handlebars
* Gulp
* Browserify
* Stylus
* Coffeescript

### Setup

* Clone this repo.
* Install and activate the required version of node.

```bash
nvm install  # install
nvm use      # activate
```

* Run `npm install`.

### Running

1. `npm start` will generate assets and start the server.
2. Browse to: `http://localhost:3002`


### Deploying

1. Create your heroku instance `heroku create`
2. Deploy `git push heroku master`
3. Dance.
